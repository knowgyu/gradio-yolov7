# gradio_yolo_inference
---
## Description
Gradio를 활용한 Yolov7 모델 웹서빙 시스템 구축입니다.

1. 모델 선택
2. Image 혹은 Video 탭 선택 후 영상 업로드
3. Run
  + Upload model 버튼을 통해 모델을 직접 추가할 수 있습니다.
  
## Usage
```
# Start
docker-compose up

# Stop
docker-compose down
```
